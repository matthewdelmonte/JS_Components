4. TypeScript Compiler (tsc)
- Written in TypeScript itself
- Compiles .ts files to .js
- Installed with NPM package (Node.js)
- Supports ES6 Syntax

Set watch mode to auto compile current file or select files
```
tsc <file.ts> -w
```
